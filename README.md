# MINI Catalyst Center IP Admin

Portfolio-ready demo for network inventory, admin access, and operational visibility. The project is intentionally clean: it uses demo hostnames, `192.168.1.1` as the demo IP, `admin/admin123` lab credentials, and no production addresses or company-specific references.

## Features

- Admin login demo using `admin/admin123!`.
- Inventory view for Catalyst-style network devices.
- Device health, software status, and access workflow mock data.
- Interactive demo actions for inventory sync, backup, and port checks.
- Static GitHub Pages demo under `docs/`.

## Demo

Open `docs/index.html` locally or publish the `docs` folder with GitHub Pages.

Target GitHub Pages source:

```text
Branch: main
Folder: /docs
```

Recommended GitHub Pages configuration:

- Source: Deploy from branch
- Branch: `main`
- Folder: `/docs`
